# KevinOS Portfolio

![KevinOS Banner](https://img.shields.io/badge/KevinOS-v4.0_Pro-blue?style=for-the-badge&logo=apple)

Bienvenido a KevinOS, un portafolio web interactivo que simula un sistema operativo de escritorio completo. Construido con tecnologias web modernas para ofrecer una experiencia de usuario unica y memorable.

## Caracteristicas Principales

### Gestion de Ventanas Avanzada
*   Resizing en 8 direcciones: Control total sobre el tamano de las ventanas desde cualquier borde o esquina.
*   Snap Zones (Estilo Windows 11): Arrastra ventanas a los bordes laterales para organizarlas automaticamente en pantalla dividida. Solo suelta la ventana cuando veas la zona azul de anclaje.
*   Arreglo Automatico: Organiza todas las ventanas abiertas en modo cascada o mosaico desde el menu contextual del escritorio.

### Spotlight Search
*   Motor de Busqueda Fuzzy: Encuentra aplicaciones, proyectos, habilidades y enlaces externos de forma instantanea.
*   Navegacion por Teclado: Atajo Universal (Ctrl + K o Cmd + K). Usa las flechas para navegar y Enter para abrir.
*   Categorias Inteligentes: Resultados agrupados por tipo (Apps, Proyectos, Links).

### Finder Pro (Explorador de Archivos)
*   Sistema de Pestanas: Soporte para multiples pestanas que permiten navegar entre diferentes carpetas simultaneamente.
*   Barra de Herramientas Dinamica: Opciones de filtrado por lenguaje (Python, JavaScript, PHP, etc.), ordenamiento (Nombre, Tipo) y modos de vista (Iconos, Lista, Miniaturas).
*   Integracion con GitHub: Sincronizacion dinamica con repositorios reales de GitHub, mostrando estrellas, lenguajes y descripciones.

### Escritorio y Personalizacion
*   Menu Contextual Enriquecido: Nuevo menu de clic derecho con submenus para gestionar el sistema.
*   Persistencia de Preferencias: El sistema guarda en localStorage tus preferencias de fondo de pantalla, tamano de iconos y ordenamiento.
*   Launchpad y Dock: Acceso rapido a todas las herramientas y aplicaciones favoritas.

## Especificaciones Tecnicas

Este proyecto ha sido construido utilizando:

*   HTML5: Estructura semantica optimizada para SEO.
*   TailwindCSS: Estilizado responsivo y moderno.
*   Vanilla JavaScript (ES6+): Motor de gestion de ventanas, sistema de pestanas y logica del sistema operativo.
*   FontAwesome: Iconos vectoriales consistentes.

## Comandos de Terminal

Abre la terminal y prueba estos comandos para interactuar con el Kernel de KevinOS:

*   ls: Listar archivos del directorio actual.
*   hackme: Ejecutar diagnostico completo de seguridad.
*   thanos: Optimizar memoria y limpiar cache del sistema.
*   destroy: Verificar integridad del disco duro.
*   cat: Concatenar y mostrar logs del sistema.
*   party: Ajustar configuracion de pantalla y color.
*   weather: Sincronizar servicios en la nube.
*   future: Calcular proyecciones del sistema.
*   linux: Informacion del Kernel.
*   ping: Comprobar latencia con el servidor.
*   clear: Limpiar la pantalla.

## Instalacion y Uso

Simplemente clona este repositorio y abre el archivo index.html en tu navegador favorito. No requiere servidores ni dependencias de Node.js para ejecutarse.

```bash
git clone https://github.com/kevorteg/portafolio.git
cd portafolio
# Abre index.html
```
