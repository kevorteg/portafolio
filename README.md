# KevinOS Portfolio 🖥️

![KevinOS Banner](https://img.shields.io/badge/KevinOS-v4.0_Pro-blue?style=for-the-badge&logo=apple)

Bienvenido a **KevinOS**, un portafolio web interactivo que simula un sistema operativo de escritorio completo. Construido con tecnologías web modernas para ofrecer una experiencia de usuario única y memorable.

## 🚀 Características Principales

*   **Interfaz de Escritorio Completa**: Ventanas arrastrables, Dock animado, Launchpad y Barra de Menú.
*   **Sistema de Archivos y Apps**:
    *   **Finder**: Navegador de archivos simulado.
    *   **Safari**: Muestra mis proyectos web.
    *   **Terminal**: Interactiva con comandos secretos (prueba `matrix` o `hack`).
    *   **Mail**: Formulario de contacto integrado.
    *   **Skills Center**: Visualización de mis habilidades técnicas.
*   **Entretenimiento**:
    *   🐍 **Snake**: Juego clásico con puntaje.
    *   🏓 **Pong**: Nuevo mini-juego contra la IA.
    *   🎵 **Música**: Reproductor de Spotify integrado.
*   **Personalización**:
    *   Fondos de pantalla dinámicos (imágenes locales).
    *   Modo Oscuro nativo.
    *   Pantalla de Bloqueo con autenticación simulada.

## 🛠️ Tecnologías

Este proyecto ha sido construido utilizando:

*   **HTML5**: Estructura semántica.
*   **TailwindCSS**: Estilizado rápido y moderno.
*   **Vanilla JavaScript (ES6+)**: Lógica del sistema, gestión de ventanas y juegos.
*   **FontAwesome**: Iconos vectoriales.

## 🕹️ Comandos de Terminal Divertidos

Abre la terminal y prueba estos comandos:

*   `help`: Muestra la lista de comandos.
*   `whoami`: Muestra el usuario actual.
*   `matrix`: Activa el efecto de lluvia de código.
*   `hack`: Simula una secuencia de hackeo.
*   `clear`: Limpia la pantalla.

## 💿 Instalación y Uso

Simplemente clona este repositorio y abre el archivo `index.html` en tu navegador favorito. No requiere servidores ni dependencias de Node.js para ejecutarse.

```bash
git clone https://github.com/kevorteg/portafolio.git
cd portafolio
# Abre index.html
```
